Modify a Markdown file into HTML

Type of challenge: learning
Duration: 60 min
challenge: solo

Learning objectives

At the end of this challenge you should be able to:

  • use basic HTML tags
  • convert MarkDown to HTML

The mission

You have to convert the MarkDown from this link: exercise into HTML inside a file called index.html
your work must be in a public repository named markdown-warmup-html.

Instructions

  • create the repository
  • convert the MarkDown to HTML
  • commit/push
